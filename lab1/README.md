Instructions for running the code on the virtual machine:
- Place the lab1.cc file in the /scratch folder within the ns-3-dev root directory
- Place the Lab1Run.sh file in the ns-3-dev directory
- Run the Lab1Run.sh script from the ns-3-dev directory
- The results.csv file will be produced in the ns-3-dev directory
- The results.csv file can be downloaded or copied onto a local machine\
- The plot_lab1.py should be run in the same directory as the results.csv file to produce all of the associated plots
